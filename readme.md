# 기능 요구 사항

1. 게임 시작 input
2. 컴퓨터가 중복 없는 세자리 수 생성
    - 숫자 인지
    - 세자리 인지
    - 중복 검사
3. 사용자가 중복 없는 세자리 수 입력
    - 숫자 인지
    - 세자리 인지
    - 중복 검사
4. 컴퓨터가 사용자 입력 숫자가 볼인 지 스트라이크 인지 판별
5. 컴퓨터가 볼인 지 스트라이크 출력
    - 볼 0 스트라이크 0 면 nothing
    - 3,4,5 반복
    - 스트라이크 3개면 6번으로
6. 축하합니다 문자 출력, 다시 시작 묻기
    - 다시 시작 y -> 2번으로
    - 다시 시작 n -> 종료

- 추가사항 : 라이프 포인트, 총 기회
